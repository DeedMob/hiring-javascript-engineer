Reduce
==========================================
Given the following:

```javascript
[
    {
        "values":[ 1, 3, 5 ]
    },
    {
        "values":[ 5, 3 ]
    },
    {
        "values":[]
    },
    {
        "values":[ 5, 3, 5, 1 ]
    }
]
```
 
Reduce the sum all of the nested values on the array of objects.

Expected answer: 31
